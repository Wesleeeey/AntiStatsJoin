# AntiStatsJoin

## Installation

* Ouvrez le terminal faite `npm i`
* Aller dans le fichier `config.json` mettez le token de votre bot, mettre un channelMove, mettre les channels BL.
* Ensuite allez dans le terminal et faite `node index.js`
